# AI Bootcamp — Misión TIC 2022 (Artificial Intelligence)

Portfolio of my work in the **Misión TIC 2022** Artificial Intelligence track (operated by Universidad Tecnológica de Pereira for MINTIC), covering neural networks, computer vision, NLP and time-series forecasting with deep learning.

## Contents

### Mission 1 — Neural Network Foundations
TensorFlow Playground experiments and first neural networks.

### Mission 2 — Core Deep Learning
- **Neural network architectures** — dense networks for regression and classification with Keras and scikit-learn.
- **Computer Vision (CNN)** — convolutional networks, handwritten-digit recognition, image classification and OpenCV.
- **NLP** — sentiment analysis on the IMDB 50K Movie Reviews dataset and recurrent neural networks.

### Mission 3 — Time Series & LSTM (capstone)
- **Bitcoin price forecasting** with an **LSTM** recurrent network on the [Kaggle Bitcoin Historical Data](https://www.kaggle.com/datasets/mczielinski/bitcoin-historical-data) dataset.
- Time-series analysis, forecasting and model validation.

## Tech
Python · TensorFlow · Keras · scikit-learn · Pandas · NumPy · Matplotlib · OpenCV · Jupyter
